3. Create Railway Account
   → https://railway.app/
   → Sign in with GitHub


🚀 DEPLOYMENT (3 Simple Steps):
═══════════════════════════════════════════════════════════════

STEP 1: Create GitHub Repository
─────────────────────────────────────────────────────────────
1. Go to: https://github.com/new
2. Repository name: discord-bot
3. Select: PRIVATE
4. DON'T check any boxes (no README, no .gitignore)
5. Click "Create repository"
6. COPY the repository URL
   (looks like: https://github.com/username/discord-bot.git)


STEP 2: Push Code to GitHub
─────────────────────────────────────────────────────────────
1. Double-click: DEPLOY.bat
2. Paste your repository URL when asked
3. Press Enter
4. Wait for "SUCCESS!" message


STEP 3: Deploy on Railway
─────────────────────────────────────────────────────────────
1. Go to: https://railway.app/
2. Click "Login" → Sign in with GitHub
3. Click "New Project"
4. Select "Deploy from GitHub repo"
5. Choose "discord-bot" repository
6. Wait for build (2-3 minutes)

7. ADD BOT TOKEN:
   - Click "Variables" tab
   - Click "New Variable"
   - Name: DISCORD_TOKEN
   - Value: (paste your bot token)
   - Click "Add"

8. Bot restarts automatically
9. CHECK LOGS - Should see: "We are ready to go in, [bot name]"
10. YOUR BOT IS NOW ONLINE! ✓

❓ TROUBLESHOOTING:
─────────────────────────────────────────────────────────────

Q: Git not recognized?
A: Install Git from https://git-scm.com/download/win
   Then restart your terminal

Q: Bot not online?
A: Check Railway logs for errors
   Verify DISCORD_TOKEN is set correctly
   Make sure token is valid in Discord Developer Portal

Q: Build failed?
A: This version has FIXED dependencies
   If it still fails, check Railway logs
   Make sure repository pushed correctly

Q: Want to start over?
A: Delete GitHub repository
   Delete Railway project
   Run DEPLOY.bat again with new repository
